Platform Choices

This project uses plain HTML and CSS hosted with GitHub Pages. This platform was chosen because it is free, stable, easy to maintain, and appropriate for a lightweight public-facing prototype.

Essential Files

The site depends on HTML files for structure, one shared stylesheet for design, and image files for the project artifact and interface sketches. These files are stored in the repository for easy reuse and revision.

Future Use

A future user could extend this site by adding more visualizations, expanding the dataset, or integrating additional historical sources. The simple file structure makes the project easy to understand and update.
