# Licorice, Land, and Power

This project is a public-facing GitHub Pages prototype for HIST 501D Digital History.

## Topic
The project examines licorice production and trade in late Ottoman Western Anatolia.

## Research Question
How did the commercialization of licorice reshape relationships between foreign companies, local actors, and the Ottoman state?

## Files Included
- `index.html` - homepage
- `findings.html` - project artifact
- `planning.html` - interface sketches and design choices
- `sustainability.html` - sustainability notes
- `styles.css` - shared stylesheet
