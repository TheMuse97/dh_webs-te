Network Visualization

This network visualization maps connections between historical actors and places in the licorice trade of late Ottoman Western Anatolia. It draws on a relational dataset organized into actors, places, and actor-place links.

Even in this early prototype, the network helps show how commercial activity depended on repeated connections between firms, individuals, and specific localities. The visualization simplifies the historical record, but it provides a useful first view of the spatial and relational structure of the project data.

Built for HIST 501D Digital History, Spring 2026.
